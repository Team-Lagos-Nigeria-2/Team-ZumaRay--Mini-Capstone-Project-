# Patient Records Management System 🏥
**SPARK Academy 2026 | Mini Capstone Project #02**

## 📝 Project Overview
This project is an interactive, command-line based management system designed for healthcare administrators. It transforms raw patient data from a CSV file into a structured system of Python objects, allowing for real-time data handling and clinical insights.

### Key Features:
* **Data Cleaning:** Automatically handles missing values in patient status records using Pandas.
* **OOP Architecture:** Utilizes `Patient` and `RecordsManager` classes to manage medical data efficiently.
* **Search & Filter:** Search for specific patients by ID/Name or filter entire wards to see patient lists.
* **Data Analytics:** Generates summary statistics (average age, ward counts) and 3 distinct visual charts (Bar, Histogram, and Pie charts).
